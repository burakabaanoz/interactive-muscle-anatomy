# Human Muscle Anatomy (Interactive SVG)

An interactive human muscle anatomy visualization built with **HTML, CSS, SVG, and JavaScript**.

- Search muscle names with autocomplete
- Highlight selected muscles on the body
- Explore major muscle groups visually
- Learn human muscle anatomy in an intuitive way
